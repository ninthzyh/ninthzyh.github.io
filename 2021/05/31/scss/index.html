scss是css的超集，最接近css写法，sass升级版，兼容css可以在scss文件中import导入css；

sass（花括号、分号）；

sass（缩进）；

3.变量

1）定义

$变量名；命名规则（字母|数字|中划线-）

$width: 1600px;
$pen-size: 3em;
2）使用

定义好的变量放在要使用的位置

#app{
    height:$width;
    font-size:$pen-size;
}
3）作用域

a.变量支持块级作用域，嵌套外的是全局变量，嵌套内是局部变量；

b.局部作用域转全局作用域，声明!global;

$width1:10em;
#foo{
    // !global将局部变量转换为全局变量
    $width:5em !global;
    // width:5em;
    width:$width;
    // heigth:10em; 使用全局变量width:10em;
    height:$width;
}
#bar{
    // width:5em; !global将局部变量转换为全局变量
    width:$width;
}

4.数据类型

数字 1，2.5，10px，5a 1.有单位也是数字；
2.不是单位的也会看作数组，能通过编译，∵scss是弱类型，可能会产生其他错误；
字符串 “foo”，‘foo’，foo 1.有引号字符串（单双引号）引号中包含空格；
2.无引号字符串，引号中不包含空格；
颜色 red，#fff，rgba(255,0,0,0.5)
布尔型 `true false`
空值 null 逻辑判断中用
数组 1.5em 1em 0 2em, Helvetica, Arial, sans-serif 用空格
映射 (key1:value1,key2:value2) 对象（maps）

判断数据类型：type-of($value)

7.杂货语法

1）插值

定义变量：$变量名：变量值；

#{变量名}用在选择器、属性名、属性值，直接塞进去赋值

$name: foo;
$attr: border;
p.#{$name} {
  #{$attr}-color: $name;
}

// 编译后：
p.foo {
  border-color: foo;
}
2）&

父选择器，&父的名字

a {
    color: yellow;
    // &:hover <=> a:hover
    &:hover{
        color: green;
    }
    &:active{
        color: blank;
    }
}
3）!default

$变量:变量值 !default;

a.之前没定义变量，$变量:变量值 !default;调用变量时，值为当前定义的变量值；

b.之前定义了一个变量，再调用变量时值为之前定义的变量；

c.定义了一个变量为null，编译后值不存在；

$isNull: null;
$content: "First content";
$content: "Second content?" !default;
$new_content: "First time reference" !default;

#main {
  isNull: $isNull;
  content: $content;
  new-content: $new_content;
}

// 编译为：
#main {
  content: "First content";
  new-content: "First time reference"; 
}
4）!global

局部变量提升为全局变量

$w:10px;

body{
 // 局部变量 !global;提升成全局变量
  $w1: 5px !global;
  width: $w;
}
p{
  width: $w1;
}
// 编译为：
body {
  width: 10px;
}

p {
  width: 5px;
}
5）!optional

没有定义选择器，@extend .select 会报错，使用!optional 可以不生成新选择器，不报错

a{
  @extend .select !optional;
}

// 编译后不报错，不执行这一行

8.@-Rules

1）@import

可以嵌套@import，不能在混合指令（minix）中嵌套；

引入scss文件，不希望被编译成css，@import _file.scss引入带下划线的scss文件；

// 导入scss，不加后缀会自动找符合名字的scss或sass文件
@import file.scss;
@import file;
// css 语法，不导入scss文件
@import file.css;
@import url('xxx');
@import 'http://xxx';
@import file screen;
// scss导入多个文件
@import 'file1','file2';
// 导入插值语句
$family: unquote("Droid+Sans");
@import url("http://fonts.googleapis.com/css?family=#{$family}");
// 编译为：
@import url("http://fonts.googleapis.com/css?family=Droid+Sans");
2）@media

css规则中嵌套

.sidebar {
  width: 300px;
  @media screen and (orientation: landscape) {
    width: 500px;
  }
}
.sidebar {
  width: 300px;
}
// 编译为
@media screen and (orientation: landscape) {
  .sidebar {
    width: 500px;
  }
}

10.混合指令（mixin）

1）定义：@mixin 混合名称
2）引用：@include 混合名称
@mixin large-text{
  font: {
    family: Arial;
    size: 20px;
    weight: bold;
  }
  color: #ff0000;
}
p{
  @include large-text;
}
// 编译后
p {
  font-family: Arial;
  font-size: 20px;
  font-weight: bold;
  color: #ff0000;
}
3）参数

@mixin(参数)

位置传参

@mixin mp($width){
  margin: $width;
}
body{
  @include mp(10px);
}
// 编译后
body {
  margin: 10px;
}

默认参数

@mixin mp($width:20px){
  width:$width;
}
body{
  // @include mp(30px);
  @include mp($width:30px);
}
// 编译后
body {
  width: 30px;
}

不确定个数的参数

不确定调用时传入了多少参数，在定义的时候可以用...表示多个参数。所有参数都放在引用的位置。

@mixin 混合名称($params...)

@mixin mp($color...){
  color:$color;
  border-color:$color;
}

body{
  @include mp(red,blue,green);
}
// 编译后
body {
  color: red, blue, green;
  border-color: red, blue, green;
}

    @content;
  }
}

@include mp{
  color: red;
  .a{
    width: 100px;
  }
}
// 编译后
body {
  color: red;
}

e. 颜色函数

  • RGB函数

    函数名和参数类型 函数作用
    rgb($red, $green, $blue) 返回一个16进制颜色值
    rgba($red,$green,$blue,$alpha) 返回一个rgba；$red,$green和$blue可被当作一个整体以颜色单词、hsl、rgb或16进制形式传入
    red($color) 从$color中获取其中红色值
    green($color) 从$color中获取其中绿色值
    blue($color) 从$color中获取其中蓝色值
    mix($color1,$color2,$weight?) 按照$weight比例，将$color1和$color2混合为一个新颜色

  • HSL函数

    函数名和参数类型 函数作用
    hsl($hue,$saturation,$lightness) 通过色相（hue）、饱和度(saturation)和亮度（lightness）的值创建一个颜色
    hsla($hue,$saturation,$lightness,$alpha) 通过色相（hue）、饱和度(saturation)、亮度（lightness）和透明（alpha）的值创建一个颜色
    saturation($color) 从一个颜色中获取饱和度（saturation）值
    lightness($color) 从一个颜色中获取亮度（lightness）值
    adjust-hue($color,$degrees) 通过改变一个颜色的色相值，创建一个新的颜色
    lighten($color,$amount) 通过改变颜色的亮度值，让颜色变亮，创建一个新的颜色
    darken($color,$amount) 通过改变颜色的亮度值，让颜色变暗，创建一个新的颜色
    hue($color) 从一个颜色中获取亮度色相（hue）值

  • Opacity函数
    alpha($color)/opacity($color) 获取颜色透明度值
    rgba($color,$alpha) 改变颜色的透明度
    opacify($color, $amount) / fade-in($color, $amount) 使颜色更不透明
    transparentize($color, $amount) / fade-out($color, $amount) 使颜色更加透明

@mixin：

1）可以传参；

2）编译完成的代码重复片段高，gzip压缩后可以很好优化重复代码，编译后不会肿胀，压缩后文件小；

@extend：

1）编译后创造DRY，gzip压缩后会破坏DRY

尽量使用@mixin！！
