media

media

1.媒体设备类型

media=”screen” // 屏幕适用

media=”print” // 打印适用

media=”screen,print” // 屏幕&打印适用

<!-- screen设备 -->
<style media="screen">
    h1{
        color: red;
    }
</style>
<!-- print设备 -->
<style media="print">
    h2{
        color: green;
    }
</style>
<body>
    <h1>media screen</h1>
    <h2>media print</h2>
</body>

2.link标签中使用media

<!-- screen适用样式 -->
<link rel="stylesheet" href="./screen.less" type="text/less" media="screen"/>
<!-- print适用样式 -->
<link rel="stylesheet" href="./print.less" type="text/less" media="print">
<script src="https://cdnjs.cloudflare.com/ajax/libs/less.js/3.12.2/less.min.js"></script>

5.and|,|not|only

and（与） screen and () orientation:landscade(横屏)
or（或） screen and(),screen and()
not（非） not screen
only（支持媒体查询的语法，就执行后面的语句） only screen 排除掉低端浏览器

两个尺寸范围重叠交集属于后写的语句

screen and (min-width:768px) // width>768

screen and (max-width:768px) // width<=768

<div class="and">and</div>
<div class="or">or</div>
<div class="not">not</div>
// 769<width<1024
@media screen and (min-width:768px) and (max-width:1024px){
    .and{
        color: red;
    }
}
// 横屏|宽度>768px
@media screen and (orientation:landscade),screen and (min-width:768px){
    .or{
        color:green;
    }
}
// width!>768
@media not screen and (min-width:768px){
    .not{
        color: blue;
    }
}

6.文件结构

文件中引入不同设备适用的less文件

a.@import url(路径可不带双引号) only screen ***;

b.引用的路径写入适用样式

width>800 //blue

width>1000 //red

c.引入的媒体查询样式按尺寸顺序从上到下执行

@import url(small.less) only screen and (min-width:800px);
@import url(middle.less) only screen and (min-width:1000px);
// small.less
body{
    background:blue;
}
//middle.less
body{
    background:red;
}

或:

@import 'small.less';
@import 'middle.less';
// small.less
@media only screen and (min-width:800px){
    body{
        background: blue;
    }
}
// middle.less
@media only screen and (min-width:1000px){
    body{
        background: red;
    }
}

        width: 100%;
        ul{

            padding: .3rem .6rem;
          }

                    border-bottom-left-radius:.3rem;
                }
                // a的最后一个元素圆角
                &:last-child{
                    border-top-right-radius:.3rem;
                    border-bottom-right-radius: .3rem;
                }
            }
        }
    }
}

12.控制边距组件

可以分别对css进行组件拆分，分别维护不同的css，边距也可类似栅格布局进行定义。

13.rem响应布局

不同尺寸的样式分别定义:root{font-size:15px}，其他尺寸使用rem，当root的字号改变rem会相对root的字号进行缩放。
