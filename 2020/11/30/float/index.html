浮动布局

浮动布局

1.文档流与浮动空间的消失

块级元素：浮动会脱离文档流，从左向右排列，不占空间位，影响后面的元素，后面的元素不浮动会补充到浮动的位置上。

行级元素：浮动会变为行级块，浮动到相应位置，依然占据空间位，能设置宽高。

<div class='box1'></div>	
<div class='box2'></div>
.box1,.box2{
  width: 100px;
  height: 100px;
}
.box1{
  border: 2px solid red;
  float: left;
}
.box2{
  background: blue;
}

2.左右浮动

浮动参考父盒子，在内边距内padding内。

<div class="main">
      <div class="box1"></div>
      <div class="box2"></div>
</div>
.main{
  width: 300px;
  height: 100px;
  border:1px solid;
  margin: 0 auto;
  padding: 10px;
  .box1,.box2{
    width: 100px;
    height: 100px;
  }
  .box1{
    border: 2px solid red;
    float: left;
  }
  .box2{
    background: blue;
    float: right;
  }
}

3.行级元素浮动后转换为行级块

行级元素没有宽高，在一行。浮动后转换为块级元素，有宽高

<div class="main">
      <span class="box1">box1</span>
      <span class="box2">box2</span>
</div>
.main{
  width: 300px;
  height: 100px;
  border:1px solid;
  margin: 0 auto;
  padding: 10px;
  .box1,.box2{
      width: 100px;
      height: 100px;
    }
    .box1{
        border: 2px solid red;
        float: left;
    }
    .box2{
        background: blue;
        // float: right;
  }
}

4.清除浮动（clear）

清除不占空间位对后面元素的影响;

a.空的块级元素清除浮动后能撑开父元素
<div class="main">
      <div class="box1">box1</div>
      <div class="box2">box2</div>
      <div class="box3">box3</div>
</div>
.main{
  border:1px solid;
  margin: 0 auto;
  padding: 10px;
  .box1,.box2{
    width: 100px;
    height: 100px;
  }
  .box1{
    border: 2px solid red;
    float: left;
  }
  .box2{
    background: blue;
    float: right;
  }
  .box3{
    clear: both;
  }
}
b.伪元素清除浮动，在父元素后面增加伪元素，通过清除一个空的伪元素并把伪元素转换成块级元素来撑开父元素；
<div class="main">
      <div class="box1">box1</div>
      <div class="box2">box2</div>
</div>
.main{
  border:1px solid;
  margin: 0 auto;
  padding: 10px;
  .box1,.box2{
    width: 100px;
    height: 100px;
  }
  .box1{
    border: 2px solid red;
    float: left;
  }
  .box2{
    background: blue;
    float: right;
  }
}
.main::after{
  content:'';
  clear: both;
  display: block;
}
c.BFC清除浮动overflow:hidden;
