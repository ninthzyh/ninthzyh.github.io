1.[页面导入样式时，使用link和@import有什么区别？]

link：

1.页面渲染同时加载，

2.xhtml，无兼容问题；

3.支持用js控制dom修改样式；

<link href='old.css' type='text/css' rel='stylesheet' id='link'>
let linkId = document.getElementById('link')
linkId.href = 'new.css'

@import：

1.页面渲染结束加载；

2.css，存在兼容问题，不兼容ie5之前；

3.不支持；

2.[[css]圣杯布局和双飞翼布局的理解和区别，并用代码实现]

圣杯布局:

1）dom直观，div包裹整个主区域；

2）相对定位；

3）代码量多，清除浮动多，副作用少；

双飞翼布局:

1）dom不直观，div包裹主区域的中间区域；

2）不用定位；

3）代码量少，清除浮动少，副作用多；

圣杯布局
header header 头
left content right div 主
footer footer 底

1）主区域的content优先加载，要将content的div放在第一个位置，此时从上到下排列为left content right,

2）float将主区域的三块内容显示在一行，设置负外边距，给main增加padding，padding左右两侧的距离是左右元素的宽度，再设置margin元素不会出去。调整顺序.content{width: 100%;}主区域的中间内容独占一行。

左区域.left{margin-left:-100%;}使左区域在中间区域的左边；

右区域.left{margin-left:-右区域宽度;}使左区域在中间区域的右边；

3）给左右区域增加相对定位，就不会挡住中间区域；

<header class="header">header</header>
    <div class="main">
      <!-- 圣杯布局先加载主区域，优先显示 -->
      <div class="content">content</div> 
      <div class="left">left</div>
      <div class="right">right</div>
    </div>
<footer class="footer">footer</footer>

  }
}
双飞翼布局
header header 头
content div content div 中
left div 左
right div 右
footer footer 底

双飞翼布局主区域中的中间区域被div包裹

1）中间区域宽度.main{width:100%}，

左区域.left{margin-left:-100%}调整左区域与中间区域的顺序，

右区域.right{margin-right:右区域的宽度}

左右区域会遮住中间区域，给中间区域包裹一层div，设置浮动和宽度。给中间区域设置外边距中间缩进；

<header class="header">header</header>
    <main class="main">
      <div class="center">conter</div>
    </main>
    <div class="left">left</div>
    <div class="right">right</div>
<footer class="footer">footer</footer>

3.[[js]用递归算法实现，数组长度为5且元素的随机数在2-32间不重复的值]

// new 长度=5的数组
let arr = new Array(5) 
let num = randomNum()
//计数器
let i = 0
randomArr(arr,num)
function randomArr(arr,num){
    //arr数组中没有随机数num，给arr索引位置赋值，i++,
    //否则重新获取随机数
    if(arr.indexOf(num)<0){
        arr[i] = num;
        i++
    }else{
        num=randomNum()
    }
    //计数器i<arr数组长度,重新调用randomArr()
    //否则return
    if(i<arr.length){
        randomArr(arr,num)
    }else{
        console.log(arr)
        return
    }
}

//2-32的随机数
function randomNum(){
    //floor向下取整，random取(0,1)随机数
    return Math.floor(Math.random()*31+2)
}
