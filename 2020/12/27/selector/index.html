选择器

1.选择器

1）通配符(*)

2）标签

3）id（唯一）&类

2.结构选择器

<main>
    <article>
        <h1>结构选择器h1</h1>
        <h2>结构选择器h2</h2>
        <aside>
            <h2>结构选择器h2</h2>
        </aside>
        <h2>结构选择器h2</h2>
        <h2>结构选择器h2</h2>
    </article>
</main>

1）所有后代

// main article 所有h2
main article h2{
  color: lightblue;
}

2）子元素

// main article子元素h2
main article>h2{
  color: lightblue;
}

3）兄弟

// main article h1的兄弟元素h2
main article h1~h2{
  color: lightblue;
}

4）铁兄弟

// main article h1紧挨下面的兄弟h2
main article h1+h2{
  color: lightblue;
}

3.属性选择器

1）属性约束，选择属性[title]

<h4 title>属性选择器</h4>
<h4>属性选择器2</h4>
h4[title]{
  background: lightblue;
}

2）属性值约束，[title=’attribute’]

<h4 title="attribute">属性选择器</h4>
h4[title='attribute']{
  background: lightblue;
}

3）^以属性值开头，[title^=’attri’]

<h4 title="attr">属性选择器</h4>
<h4 title="attri">属性选择器2</h4>
h4[title^='attri']{
  background: lightblue;
}

4）$以属性值结尾，[title$=’ibute’]

<h4 title="ibute">属性选择器</h4>
<h4 title="bute">属性选择器2</h4>
h4[title$='ibute']{
  background: lightblue;
}

5）属性包含相同属性值会被选中，[key=value]

<h4 title='attribute1'>属性选择器</h4>
<h4 title='attribute12345'>属性选择器2</h4>
h4[title*='attribute12345']{
  background: lightblue;
}

6）选择器值是一个单词会被选中，[key=value]

<h4 title='attribute12'>属性选择器</h4>
<h4 title='attribute12  com'>属性选择器2</h4>
h4[title~='attribute12']{
  background: lightblue;
}

7）|以选择器值开头，并用‘-’连接下一个内容，[key|=value]

<h4 title='attribute123'>属性选择器</h4>
<h4 title='attribute123-com'>属性选择器2</h4>
h4[title|='attribute123']{
  background: lightblue;
}

1）lv|ha|fha

:link|:visited|:hover|:active （初始|点击后|悬浮|点击中）

focus（聚焦）

<a href="#">伪类选择器</a>
<input type="text">
a{
  text-decoration:none
}
a:link{
  background: lightblue;
}
a:visited{
  background: lightgreen;
}
a:hover{
  background: red;
}
a:active{
  background: yellow;
}

input:focus{
  background: lightblue;
}
input:hover{
  background: red;
}
input:active{
  background: yellow;

2）:target|:empty|:root（目标伪类|空伪类|根伪类）

<a href="#target">目标</a>
<div></div>
<div id="target">锚点</div>
<ul>
    <li>非空</li>
    <li></li>
</ul>
div{

}

li:empty{
  display: none;

:root{
  background: lightblue;
}

5.伪类选择元素

空格：元素的后代

child要紧跟冒号前的元素，type不需要

<main>
    <article>
        <h1>首尾元素选择器</h1>
        <h2>首尾元素选择器h2</h2>
        <aside>
            <h2>首尾元素选择器h2</h2>
        </aside>
    </article>
</main>

1）:first-child选择第一个元素

:last-child选择最后一个元素

article所有后代第一个|最后一个元素

article :first-child{
    color: red;
}
 article :last-child{
  color: red;
}

article后代子元素第一个|最后一个

article>:first-child{
    color: red;
}
article>:last-child{
  color: red;
}

article子元素是h2的第一个|最后一个元素

article>h2:first-child{
  color: red;
}
article>h2:last-child{
  color: red;
}

article所有后代是h1的第一个元素

article h1:first-child {
  color: red;
}

article所有后代是h1，h1下所有后代的第一个元素

article h1 :first-child {
  color: red;
}

2）:first-of-type选择第一个类型的元素

:last-of-type选择最后一个类型的元素

article所有后代的是h2，第一个|最后一个出现的类型元素

article h2:first-of-type{
  color: red;
}
article h2:last-of-type{
  color: red;
}

3）唯一选择器:only-child

article所有后代唯一的子元素。aside->h2是唯一的孙元素，有唯一性。

h1和h2是平级关系，没唯一性。

article :only-child{
  color: red;
}

4）only-of-type选择唯一元素类型

article子元素是h2的唯一类型

article>h2:only-of-type {
  color: red;
}

6.元素编号选择

1）:nth-child(n)选择第n个元素，n从1开始，选择紧跟冒号：之前的元素

 <main>
     <article>
         <h1>元素编号</h1>
         <h2>元素编号h2</h2>
         <aside>
             <h2>元素编号h2</h2>
         </aside>
     </article>
</main>

article所有后代元素下的第2个，aside这层只有一个，所以不被选中

article :nth-child(2){
    color: red;
}

article所有子元素下紧跟article的第1个

article>:nth-child(1) {
  color: red;
}

article所有子元素h1下紧跟article的第1个

article>h1:nth-child(1) {
  color: red;
}

odd：奇数，even：偶数

列表li的索引从1开始，括号中计算的结果>0，n>=0

-n+2 n=2(结果0) n=1(1) n=0(2) 选中1、2

n+2 n=0(2) n=1(3) n=2(4) 选中2之后

:nth-child(n)

<ul>
    <li>1</li>
    <li>2</li>
    <li>3</li>
    <li>4</li>
</ul>

选择li第1、2个

ul li:nth-child(-n+2){
     color: red;
}

2）:nth-last-child(n)选择倒数第n个元素，

ul所有后代元素li，选择li倒数第1、2个

ul li:nth-last-child(-n+2){
    color:red;
}

3）article所有后代元素是h2的倒数1、2个元素，包括孙元素

<main>
    <article>
        <h1>元素编号</h1>
        <h2>元素编号.com</h2>
        <aside>
            <h2>元素编号.com</h2>
        </aside>
    </article>
</main>
article h2:nth-last-child(-n+2){
    color: red;
}

4）:nth-last-of-type(n)选择倒数第n个类型的元素

article子元素是h2，倒数第一个类型

article>h2:nth-last-of-type(1){
    color:red;
}

5）:not(n)排除选择器

ul所有后代元素为li，选择第1、2、3个元素，排除第3个，排除第一个

ul li:nth-child(-n+3):not(:nth-child(3)):not(:first-child){
  color: red;
}

7.表单伪类

:disabled|:enabled|:required|:optional|:checked（禁用|可用|必填|非必填|选中）

<label>禁用</label>
<input type="text" name="" id="" disabled />
<label>可用</label>
<input type="text" name="" id="" enabled />
<label>必填</label>
<input type="text" name="" id="" required />
<span>
    <label>非必填</label>
    <input type="text" name="" id="" optional />
</span>
<hr />
<input type="radio" name="sex" id="man" checked />
<label for="man">man</label>
<input type="radio" name="sex" id="woman" />
<label for="woman">woman</label>
input:disabled{
    background: lightblue;
}
input:enabled{
    background: blue;
}
input:required{
    background: green;
}
span>input:optional{
    background: red;
}
input:checked+label{
    color: gold;
}

8.文本伪类

1）文本

::first-letter选择文本第一个字

::first-line选择文本第一行

::after|::before选择元素后面|前面（可对添加的伪类元素修改）

<p>Lorem ipsum dolor sit amet consectetur adipisicing elit. Praesentium, eos.</p>
<div>文本伪类</div>
p{
    display: block;
    width: 200px;
}
p::first-letter{
    color: red;
}
p::first-line{
    color: greenyellow;
}
div::after{
    content: '123';
    position: relative;
    left: 200px;
    top: 40px;
}

2）自定义输入框

<div>
    <input type="text" name="" id="">
    <span></span>
</div>

        border:none;
        outline: none;
    }
    input[type='text']+span::after{
        content: '🙂';
        font-size: 10px;

    }
}
