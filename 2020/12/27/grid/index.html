grid布局

grid布局

1.栅格布局4*4

template都在开启grid布局的盒子里

display:grid;

grid-template-rows:设置行

grid-template-colums:设置列

<article>
    <div>1</div>
    <div>2</div>
    <div>3</div>
    <div>4</div>
    <div>5</div>
    <div>6</div>
    <div>7</div>
    <div>8</div>
    <div>9</div>
    <div>10</div>
    <div>11</div>
    <div>12</div>
    <div>13</div>
    <div>14</div>
    <div>15</div>
    <div>16</div>
</article>
article{
    display: grid;
    grid-template-rows: repeat(4,1fr);
    grid-template-columns: repeat(4,1fr);
    width: 400px;
    height: 400px;
    border: 1px solid;
    div{
        width: 100px;
        height: 100px;
        background: lightblue;
        // 背景到盒子的content 100-10*2-1*2=78
        background-clip: content-box;
        border: 1px solid;
        padding: 10px;
        // 元素的总宽高包含到border 100*100
        box-sizing: border-box;
    }
}

2.重复设置栅格结构

行高1，行高2….，重复x次

grid-template-rows:repeat(行数，行高);

grid-template-colums:repeat(列数，列宽);

article{
    display: grid;
    // 行高100px 50px，重复2行
    grid-template-rows: repeat(2,100px 50px);
    grid-template-columns: repeat(4,1fr);
    width: 400px;
    height: 400px;
    border: 1px solid;

    div{
        background: lightblue;
        // 背景到盒子的content 100-10*2-1*2=78
        background-clip: content-box;
        border: 1px solid;
        padding: 10px;
        // 元素的总宽高包含到border 100*100
        box-sizing: border-box;
    }
}

3.自动填充行列

行|列自动填充，每个元素宽高为num*num

grid-template-rows:repeat(auto-fill,num);

grid-template-colums:repeat(auto-fill,num);

<article>
    <div>1</div>
    <div>2</div>
    <div>3</div>
    <div>4</div>
    <div>5</div>
    <div>6</div>
    <div>7</div>
    <div>8</div>
    <div>9</div>
</article>
article{
    display: grid;
    // 自动填充行列，元素100*100
    grid-template-rows: repeat(auto-fill,100px);
    grid-template-columns: repeat(auto-fill,100px);
    width: 300px;
    height: 300px;
    border: 1px solid;
    div{
        background: lightblue;
        background-clip: content-box;
        border: 1px solid;
        padding: 10px;
        box-sizing: border-box;
    }
}

article{

    height: 150px;
    border: 1px solid;
    div{
        background: lightblue;
        background-clip: content-box;

    }
}

6.栅格编号线

行起始线：grid-row-start:num;

行终止线：grid-row-end:num;

列起始线：grid-column-start:num;

列终止线：grid-column-end:num;

<article>
    <div>1</div>
</article>
article{
    display: grid;
    grid-template-rows: repeat(3,1fr);
    grid-template-columns: repeat(3,1fr);
    width: 300px;
    height: 300px;
    border: 1px solid;
    div{
        // 行线2-3，列线2-3
        grid-row-start:2;
        grid-row-end:3;
        grid-column-start:2;
        grid-column-end: 3;
        background: lightblue;
        border: 1px solid;
        padding: 10px;
        box-sizing: border-box;
    }
}

不规则元素布局

<article>
    <div>1</div>
    <div>2</div>
    <div>3</div>
</article>
article{
    display: grid;
    grid-template-rows: repeat(2,1fr);
    grid-template-columns: repeat(2,1fr);
    width: 200px;
    height: 200px;
    border: 1px solid;
    div{
        background: lightblue;
        border: 1px solid;
        padding: 10px;
        box-sizing: border-box;
    }
    // 第一个元素行线1-3，列线1-2，将元素3挤到空白处
    div:nth-child(1){

        grid-row-end:3;
        grid-column-start: 1;
        grid-column-end:2;
    }
}

7.栅格命名

1）每行|列命名，每行列都要命名

grid-template-rows:[r1行起] num [r1行终 r2行起] num [r2行终 r3行起] num [r3行终]；

grid-template-columns: [c1列起] num [c1列终 c2列起] num [c2列终 c3列起] num [c3列终]；

<article>
    <div>1</div>
</article>
article{
    display: grid;
    grid-template-rows: [r1-start] 100px [r1-end r2-start] 100px [r2-end r3-start] 100px [r3-end];
    grid-template-columns: [c1-start] 100px [c1-end c2-start] 100px [c2-end c3-start] 100px [c3-end];
    width: 300px;
    height: 300px;
    border: 1px solid;
    div{
        background: lightblue;
        border: 1px solid;
        padding: 10px;
        box-sizing: border-box;
    }
    // 2行起-2行终，2列起-2列终
    div:first-child{
        grid-row-start:r2-start;
        grid-row-end:r2-end;
        grid-column-start:c2-start;
        grid-column-end:c2-end;
    }
}
2）重复栅格命名，行列只命名一次

n行|列，每行|列高度num，栅格线每行|列起始、终止命名为括号里的内容

grid-template-rows:repeat(n, [行起] num [行终])

grid-template-columns:(n, [列起] num [列终])

article{
    display: grid;
    // 重复3行|列等分
    grid-template-rows: repeat(3,[r-start] 1fr [r-end]); 
    grid-template-columns: repeat(3,[c-start] 1fr[c-end]);
    width: 300px;
    height: 300px;
    border: 1px solid;
    div{
        background: lightblue;
        border: 1px solid;
        padding: 10px;
        box-sizing: border-box;
    }
    // 1行起-1行终，1列起-3列终
    div:first-child{
        grid-row-start: r-start 1;
        grid-row-end: r-end 1;
        grid-column-start: c-start 1;
        grid-column-end: c-end 3;
    }
}
3）栅格线数字命名，只数栅格线数字序号

不写grid-x-start，默认偏移1格

article{
    display: grid;
    grid-template-rows: repeat(3,1fr); 
    grid-template-columns: repeat(3,1fr);
    width: 300px;
    height: 300px;
    border: 1px solid;
    div{
        background: lightblue;
        border: 1px solid;
        padding: 10px;
        box-sizing: border-box;
    }
    // 数字命名只数栅格线数字序号，行线：1-2，列线：1-4
    div:first-child{
        grid-row-start:1;
        grid-row-end:2;
        grid-column-start: 1;
        grid-column-end:4;
    }
}
4）偏移量命名

行|列偏移num：grid-x-start:span num;

栅格线命名只加grid-x-end|grid-x-start，默认偏移一格；

偏移量span同时加在start|end上

article{
    display: grid;
    grid-template-rows: repeat(3,1fr); 
    grid-template-columns: repeat(3,1fr);
    width: 300px;
    height: 300px;
    border: 1px solid;
    div{
        background: lightblue;
        border: 1px solid;
        padding: 10px;
        box-sizing: border-box;
    }
    // 行end：3，列end：4，起始start同时偏移2
    div:first-child{
        grid-row-start:span 2;
        grid-row-end:3;
        grid-column-start:span 2;
        grid-column-end:4;
    }
}
5）元素定位简写

行start：x，end：y grid-row:x/y;

列start：x，end：y grid-column:x/y

less使用grid元素定位简写会被计算，要跟偏移量同时使用；

article{
    display: grid;
    grid-template-rows: repeat(3,1fr); 
    grid-template-columns: repeat(3,1fr);
    width: 300px;
    height: 300px;
    border: 1px solid;
    div{
        background: lightblue;
        border: 1px solid;
        padding: 10px;
        box-sizing: border-box;
    }
    // 行start：1，span 1；列start：1，span 3
    div:first-child{
        grid-row: 1/span 1;
        grid-column: 1/span 3;
    }
}

eg.不规则布局(小米首页)

<article>
    <div>1</div>
    <div>2</div>
    <div>3</div>
</article>
article{
    display: grid;
    grid-template-rows: repeat(3,1fr); 
    grid-template-columns: repeat(3,1fr);
    width: 300px;
    height: 300px;
    border: 1px solid;
    div{
        background: lightblue;
        border: 1px solid;
        padding: 10px;
        box-sizing: border-box;
    }
    // 行start：1，span 3；列start：1，span 1
    div:nth-child(1){
        grid-row: 1/span 3;
    }
    // 行start：1，span 2；列start：2，span 2
    div:nth-child(2){
        grid-row:1/span 2;
        grid-column:2/span 2;
    }
    // 计算完1、2位置3被挤到行3 span 1的位置；列start：2，span 2
    div:nth-child(3){
        grid-column:2/span 2;
    }
}
6）区域命名

grid-area:行/列/行/列；

less会被计算成值，使用区域定位的命名

grid-template-rows:repeat(3,[r] 1fr);

grid-template-columns:repeat(3,[c] 1fr);

<article>
    <div>1</div>
</article>
article{
    display: grid;
    // 区域定位的命名
    grid-template-rows:repeat(3,[r]1fr);
    grid-template-columns:repeat(3,[c]1fr);
    width: 300px;
    height: 300px;
    border: 1px solid;
    div{
        background: lightblue;
        padding: 10px;
        border:1px solid;
        box-sizing:border-box;
        grid-area: r 2/ c 2/ r 3/ c 3;
    }
}
7）区域模板命名

父元素：

grid-template-rows:m行;

grid-template-colunms:n列;

grid-template-areas: m行n列;

子元素：

grid-area:对应父元素中区域的名字

eg.不规则布局(小米首页)

<article>
    <header></header>
    <nav></nav>
    <main></main>
    <footer></footer>
</article>
article{
    height: 100vh;
    width: 100vw;
    display: grid;
    grid-template-rows:60px 1fr 60px;
    grid-template-columns:60px 1fr;
    // 三行两列命名
    grid-template-areas:'header header'
    'nav main'
    'footer footer';
    header,nav,main,footer{
        padding:10px;
        box-sizing: border-box;
        background: lightblue;
        background-clip: content-box;
    }
    // 各部分区域名
    header{
        grid-area: header;
    }
    nav{
        grid-area:nav;
    }
    main{
        grid-area:main;
    }
    footer{
        grid-area:footer;
    }
}

占位符优化区域模板命名

区域模板中名字相同会被合并区域；

不同用’.’占位，每个区域是独立的空间；

article{
    height: 100vh;
    width: 100vw;
    display: grid;
    grid-template-rows:60px 1fr 60px;
    grid-template-columns:60px 1fr;
    // 三行两列命名, 引号内区域名称相同会被合并;
    // '. .'占位不会合并区域
    grid-template-areas:'header header'
    '. .'
    'footer footer';
    header,nav,main,footer{
        // 现有background才能background-clip
        background: lightblue;
        padding:10px;
        box-sizing: border-box;
        background-clip: content-box;
    }
    header{
        grid-area: header;
    }
    footer{
        grid-area: footer;
    }
}
8）区域栅格线命名

每条栅格线名称：区域模板中区域命名-start|end

<article>
      <header></header>
      <nav></nav>
      <main></main>
      <footer></footer>
    </article>
article{
    height: 100vh;
    width: 100vw;
    display: grid;
    grid-template-rows:60px 1fr 60px;
    grid-template-columns:60px 1fr;
    // 三行两列命名
    grid-template-areas:'header header'
    'nav main'
    'footer footer';
    header,nav,main,footer{
        padding:10px;
        box-sizing: border-box;
        background-clip: content-box;
    }
    // 行：header结束-footer开始
    // 列: nav开始-main结束
    header{
        background: lightblue;
        grid-row: header-end/footer-start;
        grid-column: nav-start/main-end;
    }
}

8.栅格流动

栅格横向|纵向排列 自动占满空出来的空间：grid-auto-flow: row|column dense;

<article>
    <div>1</div>
    <div>2</div>
    <div>3</div>
    <div>4</div>
</article>
article{
    width: 300px;
    height: 300px;
    border: 1px solid;
    display: grid;
    grid-template-rows: repeat(3,1fr);
    grid-template-columns:repeat(3,1fr);
    // 行排列 后面的元素自动填满空出来的空间
    grid-auto-flow: row dense;
    div{
        background: lightblue;
        padding: 10px;
        border: 1px solid;
        background-clip: content-box;
        box-sizing: border-box;
    }
    div:nth-child(1){
        grid-column: 1/span 2;
    }
    div:nth-child(2){
        grid-row:2/span 1;
        grid-column: 3/span 1;
    }
}

9.栅格对齐

<article>
    <div>1</div>
    <div>2</div>
    <div>3</div>
    <div>4</div>
</article>
1）栅格整体对齐与简写

栅格整体：

justify-content: 栅格整体主轴对齐方式;

align-content:栅格整体交叉轴对其方式;

简写：place-content:align justify;

article{
    width: 500px;
    height: 150px;
    border: 1px solid;
    display: grid;
    grid-template-columns:repeat(4,100px);
    // 栅格整体对齐
    // justify-content: space-around;
    // align-content: center;
    // 栅格整体对齐简写
    place-content: center space-around;
    div{
        height: 100px;
        background: lightblue;
        padding: 10px;
        border: 1px solid;
        background-clip: content-box;
        box-sizing: border-box;
    }
}
2）栅格内元素整体对齐与简写

栅格整体：

justify-items:栅格内元素主轴对齐方式;

align-items:栅格内元素交叉轴对齐方式;

简写：place-items:align justify; 

article{
    width: 500px;
    height: 150px;
    border: 1px solid;
    display: grid;
    grid-template-columns:repeat(4,100px);
    // 栅格内元素整体对齐
    // justify-items: start;
    // align-items:center;
    // 栅格整体对齐简写
    place-items: center start;
    div{
        height: 100px;
        background: lightblue;
        padding: 10px;
        border: 1px solid;
        background-clip: content-box;
        box-sizing: border-box;
    }
}
3）栅格元素独立控制对齐与简写

栅格独立元素内：

justify-self:栅格元素独立控制主轴对齐方式;

align-self:栅格元素独立控制交叉轴对齐方式;

简写：place-self:align justify; 

article{
    width: 500px;
    height: 150px;
    border: 1px solid;
    display: grid;
    grid-template-columns:repeat(4,100px);
    div{
        height: 100px;
        background: lightblue;
        padding: 10px;
        border: 1px solid;
        background-clip: content-box;
        box-sizing: border-box;
    }
    div:nth-child(3){
        // 栅格元素独立控制对齐
        // justify-self: end;
        // align-self: center;
        // 栅格元素独立控制对齐简写
        place-self: center end;
    }
}
