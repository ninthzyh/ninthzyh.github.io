定位布局

定位布局

1.absolute参照元素：

父元素有定位属性absolute|relative|fixed|…，子元素相对于父元素定位，如果父元素没有定位，一直向上找，找不到时对于整个页面；

2.定位设置尺寸：

没设置尺寸时定位的top|bottom|left|right会影响尺寸;

3.滚动对元素定位影响

内容在容器里发生定位，有滚动条时内容随滚动条滚动，不会相对父元素始终定位

<article>
      <section>
        <div></div>
      </section>
</article>
article {
  border: 3px solid yellowgreen;
  width: 300px;
  height: 300px;
  position: relative;
  overflow: scroll;
  section{
    background: #ddd;
    width: 100%;
    height: 600px;
    div {
      width: 100px;
      height: 100px;
      background: yellowgreen;
      position: absolute;
      // 绝对定位居中，父元素有定位属性，相对于父元素定位；
      // 父元素没有定位，向上层找有定位属性，相对于该上层元素定位，
      // 没有相对于整个页面
      top:50%;
      left: 50%;
      margin-left: -50px;
      margin-top: -50px;
    }
  }
}

4.图标定位

<ul>
      <li>
        <span>hot</span>
      </li>
</ul>
ul {
  list-style: none;
  li {
    width: 100px;
    height: 100px;
    background: lightblue;
    border: 10px solid lightgreen;
    position: relative;
    box-sizing: content-box;
    span {
      color: #fff;
      background: red;
      display: inline-block;
      width: 30px;
      height: 30px;
      text-align: center;
      line-height: 26px;
      border-radius: 50%;
      font-size: 10px;
      position: absolute;
      top: -20px;
      left: -20px;
    }
  }
}

5.定位层级

a.有定位属性时，鼠标移入改变元素层级

 <ul>
      <li>
        <span>hot</span>
        <div class="first">first</div>
      </li>
</ul>
ul {
  list-style: none;
  li {
    width: 100px;
    height: 100px;
    background: lightblue;
    border: 10px solid lightgreen;
    position: relative;
    box-sizing: content-box;
    cursor: pointer;
    span {
      color: #fff;
      background: red;
      display: inline-block;
      width: 30px;
      height: 30px;
      text-align: center;
      line-height: 26px;
      border-radius: 50%;
      font-size: 10px;
      position: absolute;
      top: -20px;
      left: -20px;
      z-index: 1;
    }
    .first{
      background:red;
      width: 100px;
      height: 100px;
      position: absolute;
      color:#fff;
      z-index: 2;
    }
  }
  // 鼠标移入li>.first层级改变
  li:hover .first{
    z-index: -1;
  }
}

b.购物车

鼠标移入展开，移除收缩

<article>
      <div class="top">shop</div>
      <div class="bottom">showShop</div>
</article>
article {
  width: 100px;
  margin: 0 auto;
  position: relative;
  > div {
    border: 1px solid #ddd;
    height: 50px;
  }
  .top {
    position: relative;
    cursor: pointer;
    // 设置定位属性，增加top层级，让top层级覆盖bottom，默认层级为0
    z-index: 1;
    background: #fff;
  }
  // 默认bottom隐藏，top有border-bottom
  .bottom {
    width: 200px;
    // 绝对定位，right：0，bottom靠右，相对父级article定位
    position: absolute;
    right: 0;
    top:50px;
    display: none;
  }
}
// 鼠标移入article top的border-bottom消失，bottom出现
article:hover .top {
  border-bottom: none;
}
article:hover .bottom {
  display: block;
}

6.粘性定位sticky

a.sticky的h2在同级，新来的元素会叠加之前的粘性元素；

<article>
      <section>
        <h2>title1</h2>
        <p>p1</p>
        <h2>title2</h2>
        <p>p2</p>
      </section>
</article>

b.sticky的h2在不同级，新来的元素会把之前的粘性元素顶走；

<article>
      <section>
        <h2>title1</h2>
        <p>p1</p>
      </section>
      <section>
        <h2>title2</h2>
        <p>p2</p>
      </section>
</article>
article {
  width: 300px;
  height: 100px;
  overflow: auto;
  border:1px solid;
  position: relative;
  p{
    height: 200px;
  }
}

article section:nth-of-type(even) h2{
  background: yellowgreen;
}
article section h2{
  background: lightblue;
  // 粘性定位 top：0定位到顶部
  position: sticky;
  margin: 0;
  top:0;
}

7.固定定位fixed

a.两个元素都有定位，后出现的定位权重高，层级大于前面的定位元素，能滚动时会覆盖前面的元素

<!-- main在header后，两个元素都有定位属性 -->
<header></header>
<main></main>
header{
  height: 100px;
  width: 100%;
  // 固定定位fixed，top:0
  position: fixed;
  top:0;
  background: lightgreen;
}
main{
  height: 100vh;
  width: 100%;
  background: lightblue;
  position: relative;
  margin-top: 120px;
}

b.后面的元素没有定位属性，前面的元素能够固定

<!-- main在header后，两个元素都有定位属性 -->
<header></header>
<main></main>
header{
  height: 100px;
  width: 100%;
  position: fixed;
  top:0;
  background: lightgreen;
}
main{
  height: 100vh;
  width: 100%;
  background: lightblue;
  // position: relative;
  margin-top: 120px;
}
