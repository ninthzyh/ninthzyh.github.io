1.[[html]HTML全局属性(global attribute)有哪些（包含H5）？]

全局属性
class 类
id id，唯一
title 显示tooltip
lang 浏览器语言识别
style css
accesskey 快捷键alt+‘ ’ （href=‘//’当前浏览器用http|https协议，自动加载相应协议脚本）<a href='//baidu.com' accesskey='c'>baidu</a>
hidden 元素隐藏 <div hidden>hidden</div>
draggable 元素拖动 draggable=’false|true|auto’
dropzone 在元素上拖动数据是否拷贝|移动|链接指向（大部分浏览器不支持）dropzone=‘copy|move|link‘
dir 文本方向从左到右|从右到左 dir=’ltr|rtl’
translate 是否翻译文本（大部分浏览器不支持） translate=’yes|no‘
contenteditable 元素是否可编辑contenteditable=’false|true‘
data-* 自定义属性str ，嵌入自定义数据
spellcheck 是否对元素语法检查
contextmenu 右键点击元素出现上下文菜单（只有火狐支持）
tabindex tab控制导航次序

2.[[css]在页面上隐藏元素的方法有哪些？]

1.opacity（css）

opacity:0;

2.display（css）

display:none|block;

3.z-index（css）

4.transform（css）

缩放

transform:scale(0)

5.visibility:hidden|visible;

6.hidden（html）

3.[[js]去除字符串中最后一个指定的字符]

substr：str.substr(start,length);

1）截取str，不传参数，返回全部。不改变原始str；

2）省略stop，截取从开始到结尾；

3）start可为负数， -1：最后一个。length<0，返回空；

substring：str.substring(start,end);

1）截取str，不传参数，返回全部。不改变原始str；

2）省略stop，截取从开始到结尾；

3）如果start、end<0，把值转换成0;

4）start>end时，调换start和end的位置；

slice：str.slice(start,end);

1）截取str/arr，不传参数，返回全部。不改变原始str；

2）省略stop，截取从开始到结尾；

3）start、end可为负数， -1：最后一个；

4）start>end时，返回空；

end<0|start<0时，newend=length-end|newstart=length-start；

let s='123456';
s.substr(0,s.length-1);
s.substring(0,s.length-1);
s.slice(0,str.length-1);
